# Sports Tournament Management System

This repository contains a university-level MVP for a Sports Tournament Management System built from the project PDF and the frozen MVP scope.

## Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB + Mongoose
- Authentication: JWT + bcrypt

## Features

- User registration and login
- Role-based access for admin, manager, and spectator
- Tournament management
- Team management and tournament registration
- Match scheduling for league and knockout formats
- Score entry and automatic standings recalculation
- Public viewing of tournaments, matches, and standings
